# 구현 기능
- 지금의 페이지들

# 구조
- 첫페이지
- 메인 :월별 아카이브 출력(테그 포함)
- 상단(고정) : About(사람모습)/LINK(신규,사슬모습)/TAG(테깅)/micro(말풍선)/검색창(돋보기)
- About : 소개글 (계층구조로 예전 소개글 연결)
- Link : 특이한 성향 배제된 흥미로운 링크들
- Tag : 테크목록 출력 (예시:http://www.halryang.net/tags/)
- 기타 이하 생략
